Welcome to the Loader Tutorial

By Ben Abroms

In this tutorial, we will learn how to make the spinning loading wheel as seen on smartphone apps and websites.

The loader above is the one we will be creating for this tutorial.

First, we will need to make a div object. Let's name it loader.

<div class="loader"></div>

We don't need anything content or text because loaders are made completely out of CSS code.

Next, we'll need to make a style in CSS for the design of the loader and the animation that will make it spin.

.loader {
border: 16px solid #f3f3f3;
border-radius: 50%;
border-top: 16px solid #3498db;
width: 120px;
height: 120px;
}

We will also need to add two important animation properties to the loader code in order for it to run.

-webkit-animation: spin 2s linear infinite;
animation: spin 2s linear infinite;

This code allows the loader to define the movements of the animation.

Now we need to create two animation CSS codes that correspond to the code in the loader.

@-webkit-keyframes spin {
0% { -webkit-transform: rotate(0deg); }
100% { -webkit-transform: rotate(360deg); }
}
@keyframes spin {
0% { transform: rotate(0deg); }
100% { transform: rotate(360deg); }
}

These two CSS properties are linked to the ones we made in the loader code. Using -webkit-transform, we are able to rotate the loader itself using keyframes to give the illuison that the blue part is rotating.

Here is the final result! Your brand new loading wheel!
